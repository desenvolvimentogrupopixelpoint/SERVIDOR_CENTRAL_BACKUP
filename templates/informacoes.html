Guia Completo do PixelPlay

Bem-vindo ao PixelPlay Gerenciador de Mídias, um sistema desenvolvido para simplificar o gerenciamento de mídias em telas remotas. Com uma interface intuitiva e recursos avançados, este guia ajudará você a aproveitar ao máximo todas as funcionalidades.

Funcionalidades Principais

  • Gerenciamento de Grupos: Crie, edite ou exclua grupos para organizar seus dispositivos de forma eficiente.
  • Configuração de Dispositivos: Adicione dispositivos ao sistema com endereços IP específicos, utilizando redes como Tailscale.
  • Envio de Mídias: Faça upload de vídeos ou imagens, definindo horários de início e término para exibição.
  • Monitoramento em Tempo Real: Verifique o status dos dispositivos e mídias para garantir que tudo esteja funcionando corretamente.
  • Personalização: Configure preferências como formato de data, cores do tema e outros ajustes.

Passo a Passo de Uso

1. Criar um Novo Grupo

Na página inicial, clique no botão "Adicionar Novo Grupo":

Preencha os campos obrigatórios como nome do grupo e endereço IP, depois clique em "Criar". O grupo será exibido na tabela com seu status.

2. Adicionar Dispositivos

Acesse um grupo e clique no botão "Adicionar Novo Dispositivo". Preencha os dados do dispositivo, como:

  • Nome do Dispositivo: Ex: "Tela Entrada".
  • Endereço IP: Ex: "192.168.0.101".

3. Enviar Mídias

No grupo desejado, utilize a tabela abaixo para simular o envio de mídias. Esta tabela é apenas demonstrativa e não possui funcionalidade:

Nome da Mídia Tamanho (MB) Início Fim Ações
Promoção Semana 1 15.2 2025-01-12 08:00 2025-01-19 22:00
Promoção Semana 2 18.5 2025-01-20 08:00 2025-01-26 22:00

4. Monitorar Status

Verifique os ícones na tabela de dispositivos para acompanhar o status:

  • Verde: Dispositivo ativo.
  • Cinza: Dispositivo inativo.

Dicas Importantes

  • Use vídeos em formato .MP4 e imagens em PNG/JPEG para compatibilidade máxima.
  • Certifique-se de que os dispositivos estão conectados à internet.
  • Faça backups regulares das configurações do sistema.
